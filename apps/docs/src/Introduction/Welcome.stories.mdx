# Digital products made easy

Lambda Curry is a full-service team that collaborates with you and makes it simple to create **innovative experiences** and **impactful products**.

Our **Medusa Forms** library provides a comprehensive collection of controlled form components designed specifically for Medusa Admin and Medusa UI applications. Built with modern React patterns, TypeScript, and seamless integration with React Hook Form.
